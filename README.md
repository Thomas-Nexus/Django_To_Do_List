# Django_To_Do_List
A Django Built To-Do-List Web Application


## Technologies

## Languages:

    - Django 3.1.3 (Python 3.8.3)
    - HTML5
    - CSS

## Editor:

    - PyCharm/Command
    - Command Line
    
## User Features:

    - Create task items
    - Update item status/change item details
    - View completed/striked out items
    - Remove items once completed
